# Wallet Tracking Research Module

This folder is reserved for wallet and leaderboard research only.

Allowed:

- Pull active trader wallets from public data.
- Rank observations by PnL, volume, and recent activity.
- Store observations for later review.
- Generate strategy hypotheses.

Not allowed:

- Directly trigger live trades.
- Bypass the shared strategy interface.
- Bypass the risk manager or kill switch.
- Trade only because a whale or leaderboard wallet traded.

Any production strategy must pass research, backtesting, paper trading, and live incubation gates before real size is considered.
